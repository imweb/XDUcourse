基础代码

//图片应该含有src和alt属性
//如果图片宽高确定应设置width和height以避免重排
<img wdith="" height="" src="" alt="" />

//html5 figure & figcaption
<figure>
    <img src="images/5.jpg" width="310" height="362" />
    <figcaption>我是图标标题</figcaption>
</figure>

//响应式图片定义
<img src="images/m-256.jpg" srcset="images/s-128.jpg 128w, images/m-256.jpg 256w, images/l-512.jpg 512w" sizes="(max-width: 320px) 100vw, 128px" alt="">

//参考：http://css-tricks.com/video-screencasts/133-figuring-responsive-images/

//背景图片background-image,background-color,background-position,background-repeat组成
//最好设置一个默认的背景色，和图片颜色相似或反衬文字色，以避免网络比较慢图片下载超慢情况下出现一片空白
.class{
	background:url(...) no-repeat x y #fff;
}
//css3多背景图，第一个的层级最高，依次类推，请注意背景色设置在最后一个背景图上，不然会覆盖后面的背景图
.class{
	background:url(img1.png) no-repeat x y #fff,url(img2.png) no-repeat x y,url(img3.png) no-repeat x y #fff;
}
