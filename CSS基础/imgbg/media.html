响应式图片

响应式图片输出方法

响应式图片的输出方法：

  • 后端判断输出（见参考资料的Adaptive）
  • 万能的一张图片，然后通过css控制其大小，直接使用max-width为100%或使用media queries判断设置其大小
  • 结合js来加载不同大小的图片
  • 使用最新的srcset方案

响应式背景图片控制方法：

  • 万能的一张图片，直接使用background-size:cover或通过media queries设置其background-size
  • 通过media queries设置不同的background-image
  • 使用image-set属性

核心代码

//一刀切
img{
	max-width:100%;
	height:auto;
}
//srcset
<img src="images/m-256.jpg" srcset="images/s-128.jpg 128w, images/m-256.jpg 256w, images/l-512.jpg 512w" sizes="(max-width: 320px) 100vw, 100%" alt="">

//使用media queries加载不同的背景图片
@media screen and (max-width: 560px){
	.img-query{
		background: url(images/7-small.jpg) no-repeat center top;
		height: 143px;
	}
}
@media screen and (max-width: 1024px){
	.img-query{
		background: url(images/7-medium.jpg) no-repeat center top;
		height: 285px;
	}
}
@media screen and (min-width: 1025px){
	.img-query{
		background: url(images/7.jpg) no-repeat center top;
		height: 448px;
	}
}

//使用image-set
.demo {
    background-image: url(assets/no-image-set.png); 
    background-image: -webkit-image-set(url(assets/test.png) 1x,
           url(assets/test-hires.png) 2x);
    background-image: -moz-image-set(url(assets/test.png) 1x,
           url(assets/test-hires.png) 2x);
    background-image: -o-image-set(url(assets/test.png) 1x,
           url(assets/test-hires.png) 2x);
    background-image: -ms-image-set(url(assets/test.png) 1x,
           url(assets/test-hires.png) 2x);
    width:200px;
    height:75px;
}
