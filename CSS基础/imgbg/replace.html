图片替换文字

一些不新不旧的方法

//text-indent优化方法
.hide-text {
    text-indent: 100%;
    white-space: nowrap;
    overflow: hidden;
}

//font及color方法，ie6会有一个小点
.ir {
    font: 0/0 a;
    text-shadow: none;
    color: transparent;//原生的ie6,7,8不支持color设置为transparent
}

//before方法，不支持ie6,7
.nir {
   height:100px; /* height of replacement image */
   width:400px; /* width of replacement image */
   padding:0;
   margin:0;
   overflow:hidden;
}	
.nir:before {
   content:url(image.gif);
   display:inline-block;
   font-size:0;
   line-height:0;
}

总结

如果你直接从ie9开始，那么可以直接color:transparent;就可以；如果要兼容ie8+，那么可以直接考虑用before和content；如果你要兼容ie7+，那么可以考虑使用font: 0/0 a;；当然如果要考虑ie6+那就是最纠结的了，就得考虑使用line-height或者text-indent了，然后设置个overflow为hidden

下面这段代码兼容ie6+，当然如果你不喜欢用line-height，可以使用text-indent，如果你考虑重复使用那么text-indent的值可以使用百分比或者一个比较大的值

// CSS image replacement
// 原生7,8不支持color的transparent
// 高度不能超过字体大小乘以99（line-height）
// 使用的时候请确定元素可以设置宽度或高度，即非inline元素
.ir{
  color: transparent;
  text-shadow: none;
  background-color: transparent;
  border: 0;
  line-height:99;
  overflow:hidden;
}
