内容区域

要点解析：

  1. 布局结构先分为上中下：header#header+#container+footer#footer
  2. 再对container分为左中右： #contianer>section#main+aside#aside_left.aside+ aside#aside_right.aside
  3. css方面先设置body的display为flex，然后设置方向为垂直方向(为了拥有sticky footer效果，我们得设置html的高度为100%和body的min-height为100%)；然后再对contianer设置display为flex，方向为水平方向（默认就是水平方向）；接着设置左右边栏的大小及与container的margin，接着设置main为剩下的宽度，这可以用flex:1来达到这个目的。为了测试main的弹性宽度，你可以改变left，right的宽度，或者直接砍掉left或right，main就会弹性布满剩余的宽度。
  4. 默认aside和main是等高的，如不需要可单独设置某个的align-self值，如该demo的左边栏align-self为flex-start
  5. 为了使padding和border不影响设置的width，我们事先设置所有的元素的box-sizing为border-box
